# 連鎖構成フォーマット

連鎖がどのように構成されているかを端的に表すためのフォーマットです（独自記法）。

## 形式

```
[連鎖数][同時消し構成]:[連鎖消去点]
```

- 連鎖数: 連鎖の回数を表す数字。1連鎖（単発）で「1」、2連鎖で「2」、...、19連鎖で「19」と表記される。
- 同時消し構成: 同時消しの構成を表す文字列（後述）。
- 連鎖消去点: 連鎖によって得られる点数を表す数字。詳細は [ぷよぷよ通のルール](/docs/puyo-rules.md) を参照。

## 同時消し構成の表記

ぷよぷよ通ルールにおいては、連鎖の最後と、その1つ前のぷよの消え方が連鎖消去点、ひいては相手に送るおじゃまぷよの数に大きく影響します。
そのため、これらを端的に表すための記号を導入します。

### 最後から2番目が単色消しの場合

最後から2番目のぷよが単色消しの場合は、最後のぷよの消え方のみを表記します。

- 単色消し: （なし）
- 2色消し: `D`（Double）
- 3色消し: `T`（Triple）
- 4色消し: `Q`（Quadruple）
- 5色消し: `M`（Multiple）

また、5つ以上の連結（多連結）を含む場合は、末尾に`+`を付けます。

### 最後から2番目が複数色消しの場合

最後から2番目のぷよが複数色消しの場合は、最後のぷよの消え方に加えて、最後から2番目のぷよの消え方も表記します（`[最後から2番目のぷよの消え方][最後のぷよの消え方]`）。
例えば、最後から2番目のぷよが2色消しで、最後のぷよが3色消しの場合は、`DT`と表記します。
特別に、最後のぷよが単色消しの場合は、Singleの`S`を用いて`DS`などと表記します。

最後から2番目のぷよが複数色消しかつ多連結を含む場合は、同様に`+`を付けます。
例えば、最後から2番目のぷよが2色消し（多連結）、最後のぷよが3色消しの場合は `D+T` と表記します。
どちらも多連結の場合は `D+T+` と表記します。

## 例

- `1:40` - 単発で40点（40×1=40）
- `1+:100` - 単発で100点（5連結で消した場合：50×(0+2+0)=100）
- `1D:240` - 単発で240点（通称**1ダブ**：80×(0+0+3)=240）
- `1D+:2380` - 単発で2380点（7連結と10連結の2色消し、通称**不発弾**：170×(0+(4+7)+3)=2380）
  - 5連結以上であれば、いくつ連結していても`+`で表記する
- `2D:920` - 2連鎖で920点（通称**2ダブ**：40×1+80×(8+0+3)=920）
- `2T:1720` - 2連鎖で1720点（通称**2トリ**：40×1+120×(8+0+6)=1720）
- `4:2280` - 4連鎖で2280点（40×1+40×(8+0+0)+40×(16+0+0)+40×(32+0+0)=2280）
- `4:2340` - 4連鎖で2340点（50×(0+2+0)+40×(8+0+0)+40×(16+0+0)+40×(32+0+0)=2340）
  - 1連鎖目が5連結になっているが、最後および最後から2番目のぷよが単色消しのため構成表記なし
- `4:2840` - 4連鎖で2840点（40×1+80×(8+0+3)+40×(16+0+0)+40×(32+0+0)=2840）
  - 2連鎖目が2色消しだが、3連鎖目および4連鎖目が単色消しのため、こちらも構成表記なし
- `4:2540` - 4連鎖で2540点（40×1+40×(8+0+0)+50×(16+2+0)+40×(32+0+0)=2540）
  - 3連鎖目は5連結だが単色消し、4連鎖目も単色消しのため構成表記なし
- `4+:2960` - 4連鎖で2960点（40×1+40×(8+0+0)+50×(16+2+0)+50×(32+2+0)=2960）
  - 3連鎖目は5連結だが単色消し、4連鎖目も5連結単色消しのため、構成表記は最後のぷよの消え方のみを表す`+`となる
- `4DS:3160` - 4連鎖で3160点（40×1+40×(8+0+0)+80×(16+0+3)+40×(32+0+0)=3160）
  - 3連鎖目が2色消しで最後のぷよが単色消しのため、`DS`と表記
- `4D+S:3530` - 4連鎖で3530点（40×1+40×(8+0+0)+90×(16+2+3)+40×(32+0+0)=3530）
  - 3連鎖目が2色消し（多連結）で最後のぷよが単色消しのため、`D+S`と表記
- `4D+S+:3950` - 4連鎖で3950点（40×1+40×(8+0+0)+90×(16+2+3)+50×(32+2+0)=3950）
  - 3連鎖目が2色消し（多連結）で最後のぷよは5連結の単色消しのため、`D+S+`と表記
- `4D:3800` - 4連鎖で3800点（40×1+40×(8+0+0)+40×(16+0+0)+80×(32+0+3)=3800）
  - 3連鎖目が単色消しで最後のぷよが2色消しのため、`D`と表記
- `4DD:4680` - 4連鎖で4680点（40×1+40×(8+0+0)+80×(16+0+3)+80×(32+0+3)=4680）
  - 3連鎖目が2色消しで最後のぷよも2色消しのため、`DD`と表記
- `4D+D:5050` - 4連鎖で5050点（40×1+40×(8+0+0)+90×(16+2+3)+80×(32+0+3)=5050）
  - 3連鎖目が2色消し（多連結）で最後のぷよが2色消しのため、`D+D`と表記
- `4DD+:5210` - 4連鎖で5210点（40×1+40×(8+0+0)+80×(16+0+3)+90×(32+2+3)=5210）
  - 3連鎖目が2色消しで最後のぷよは2色消し（多連結）のため、`DD+`と表記
- `4D+D+:5580` - 4連鎖で5580点（40×1+40×(8+0+0)+90×(16+2+3)+90×(32+2+3)=5580）
  - 3連鎖目が2色消し（多連結）で最後のぷよも2色消し（多連結）のため、`D+D+`と表記
- `19:175080` - 19連鎖で175080点（すべて4連結単色消し：40×1+40×(8+0+0)+...+40×(512+0+0)=175080）
